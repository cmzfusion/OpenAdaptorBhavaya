The Persistence Layer - Property Change Events

Bhavaya's real-time functionality is provided by firing PropertyChanges to registered PropertyChangeListeners. As such Bhavaya beans must implement the org.bhavaya.util.Observable interface and the PropertyChange methods described below. However, this can be achieved easily by making any new beans a subclass of org.bhavaya.util.DefaultObservable.

  The Observable interface

public void addPropertyChangeListener(PropertyChangeListener l)
public void addPropertyChangeListener(String propertyName, PropertyChangeListener l)
public void removePropertyChangeListener(PropertyChangeListener l)
public void removePropertyChangeListener(String propertyName, PropertyChangeListener l)
