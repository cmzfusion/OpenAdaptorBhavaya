The Persistence Layer - ListEvent class

ListEvent has three properties of interest, described in the table below and retrieved by calling e.g. getRow().

property return type description
row int the row index for the value that was added, removed or updated
type int the type of the change to the collection and can be ListEvent.[INSERT||UPDATE||DELETE||COMMIT]
value object the bean that was affected (added, deleted or updated)
