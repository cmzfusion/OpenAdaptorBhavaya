The Persistence Layer

The Bhavaya persistence layer is configured via .xml files, namely application.xml, schema.xml and criterion.xml. There is a step-by-step guide to the entries required in these files, together with example source code that tests each of the new features introduced. Alternatively, complete annotated sample configuration files are available as a guide for the more adventurous.
