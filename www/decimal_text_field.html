Decimal Text Field

The DecimalTextField (org.bhavaya.ui.DecimalTextField) is an extension of the JTextField. It is intended as a number entry field, and a few of the character keys can be used as shortcuts to multiply the number entered.

The number keys operate as normal, the special keys are summarised in the table below.

KeyDescriptionmultiplier
kThousand103 (1000)
tThousand103 (1000)
mMillion106 (1000000)
bBillion (US)109 (1000000000)
,CommaDecimal point separator
.Full stop / PeriodDecimal point separator

A demo of the DecimalTextField can be run directly via JavaWebStart, or alternatively executed via its source code, or if you have downloaded and extracted examples.zip it can be run with: ant decimaltextfielddemo. There is a screen shot of the application below.

The keystrokes entered into the above DecimalTextField were 12.3k The field automatically changed its contents to numeric format.

Simple use: (see the demo source code)

      String format = "###.##";
      dtf = new DecimalTextField(format);
      JPanel contents = new JPanel();      
      contents.add(dtf);

Constructor summary:

      DecimalTextField(String format) 
      DecimalTextField(String format, int columns) 
      DecimalTextField(String format, int columns, Number initalValue)

The format string is a DecimalFormat format specifier whose use is described at the Sun Tutorial. columns sets the number of columns available in the field, and the intialValue is the initially displayed entry.

(When using this component remember to add Bhavaya's jar files to your classpath (extract_location/bhavaya/bin/bhavaya.jar and extract_location/bhavaya/lib/log4j-1.2.15-patched.jar).)
