Bhavaya Overview

The functionality provided by the complete Bhavaya package is introduced here, where its use as a real-time database viewer is described. Bhavaya encapsulates a persistence layer and a GUI optimised for displaying data that changes. These features and others are discussed in the architecture section, which covers the software's components and is intended for developers who want to use just some of Bhavaya's features.

Bhavaya demo

The Bhavaya demo application showcases the software's main features. It is self-contained and can be launched directly using JavaWebStart. A brief guide to using the demo application is given in the Quickstart section.

Screen shot of the Bhavaya demo application

The Data Collections tab on the left shows the classes that are available from the database. Bhavaya can be configured to synthesise and display classes from the database automatically (according to its contents) or in conjunction with requirements specified by the user. The classes synthesised in the demo application are from a database containing information about bond trades. The data collections tab allows criteria to be specified that determine which objects of these classes (obtained from the database data) are displayed in the spreadsheet-like area in the centre of the application. Spreadsheet cells are flashed when their corresponding database data changes. The right-hand Edit tab allows the user to specify which properties of the class (synthesised from database columns) are displayed. For more information about the persistence layer and Bhavaya's class synthesis click here. (The demo application contains an in-memory database whose content is changing in real-time as the program runs, simulating a real-world database connection.)

There is an undocked spreadsheet tab in the lower left-hand corner. This tab shows an aggregated view of the data, and highlights the application's sorting and grouping functions. Each row displays the total value of all bonds that mature within a specific time frame.

Bhavaya, used in this fashion, provides a real-time database viewer. MS Excel or MS Access users will recognise some of Bhavaya's features as part of the functionality available in database browsers and spreadsheets. Without writing any code, you can point Bhavaya at a database, select a table, load the data from the table into a grid widget in a spreadsheet-like application and then sort, group, pivot and graph it. What is different about Bhavaya is that, once you have created this configuration it is possible to use Bhavaya as a real-time application - i.e. you can track the changes to the data as they are happening in the database. This solves one of the perennial problems of database browsers and spreadsheets - that they tend to "snapshot" the data they display, and so need a system to be built around them to keep them up-to-date. Bhavaya does this automatically, by monitoring either the logs or audit trail of the database.

Bhavaya Builder

The Bhavaya Builder application (launches via JavaWebStart) provides a simple interface for creating a real-time view of database data using Bhavaya and its GUI. It is not necessary to write any code to get a view of the database data up and running, allowing non-developers to browse the data in their database. However, users will need to know the jdbc connection parameters for their database and technical work is required to set up the notification mechanism from the database to enable the real-time functionality. Quickstart instructions for using Bhavaya Builder can be found here.
