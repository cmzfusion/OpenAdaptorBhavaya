Quickstart Guides


These guides should quickly enable you to use Bhavaya or any of its features.

  • The Demo Application Showcases the software in its entirety. Completely self-contained.
  • The Builder Application A GUI application for configuring Bhavaya to connect to your database, then providing a real-time GUI view of the data in that database.
  • The Persistence Layer How to use Bhavaya's persistence layer as part of your application.
